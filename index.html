How to use

  1. Log into soundcloud.com in the same browser you're using right now.
  2. Paste a track or playlist URL below.
  3. Hit Download. Files save as [Artist] Title.mp3 in ./downloads.

The tool works for any track you can play in your browser. DRM-locked tracks (most major-label catalog) automatically fall back to YouTube via yt-dlp; those files are prefixed [YouTube].
